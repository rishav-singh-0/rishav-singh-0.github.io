ssh audit

May 20, 2025 · 1 min

Overview#

ssh-audit is a tool for ssh server & client configuration auditing(banner, key exchange, encryption, mac, compression, compatibility, security, etc).

Installation#

  • On Arch
pacman -S ssh-audit
  • pip
pip3 install ssh-audit
  • Generic - Download and extract tarball from release

Usage#

./ssh-audit.py 192.168.101.201
